NP-Complete Sudoku Solver

Python

A command line base sudoku program which allows the users to play sudoku, check their answers, or have the computer solve it for them. This program was a look at NP-Complete puzzle problems and involves a solution algorithm with a ridiculous runtime complexity of O(n^2k).

You can run Sudoku on repl.it or view it on GitHub.
